# 中文 AI 套话词汇黑名单

按"违规严重程度 + 出现频次"排序。看到表里的词，先问一句：删掉它意思变了吗？没变就删。

---

## 1. 开场套话（破题句）

中文 AI 最爱用宏大背景开头，仿佛在写高考作文。

| 套话 | 问题 |
|---|---|
| 在当今 XX 的时代 / 背景下 | 任何内容都套得上，等于没说 |
| 随着 XX 的不断发展 | 套话开头，下一句必是空话 |
| 在这个信息爆炸的时代 | 比喻磨损严重 |
| 在科技日新月异的今天 | 标准 AI 起手 |
| 众所周知 / 不可否认 / 显而易见 | 真的众所周知就不用写 |
| 自古以来 / 自人类文明诞生之初 | 强行拉历史纵深 |
| 在 XX 蓬勃发展的浪潮中 | "浪潮" 二字基本都是 AI |
| 21 世纪是 XX 的世纪 | 八股开头 |
| 当我们谈论 XX 的时候 | 没必要的预告 |

**改写思路**：第一句直接是观点、事实、场景或冲突，不要背景铺垫。

---

## 2. 八股连接词（过渡套话）

整段全是连接词撑骨架，删了之后文章一样成立。

| 套话 | 用法 | 改写方向 |
|---|---|---|
| 首先 / 其次 / 再者 / 最后 | 分点骨架 | 直接讲，不用"首先" |
| 综上所述 / 总而言之 / 总的来说 | 结尾收束 | 删掉，让最后一句自然成为结尾 |
| 概括而言 / 归根结底 | 总结 | 同上，删 |
| 由此可见 / 不难看出 / 显而易见 | 推论 | 删，或换成具体推理 |
| 与此同时 / 与之相对 | 并列 | 直接换行另起一段 |
| 一方面 …… 另一方面 …… | 二分 | 拆成两段或换句式 |
| 不仅如此 / 更重要的是 | 递进 | 删，让递进靠内容自身 |
| 然而 / 因此 / 此外 / 同时 | 高频连接 | 减半，能省就省 |
| 接下来 / 接下来让我们 | 预告 | 删，直接进入内容 |
| 在此基础上 / 基于此 | 学术腔过渡 | 改成具体动作描述 |

## 3. 强调虚壳（看似有力其实空的开头）

这些短语后面 99% 跟一句没营养的话。

- 值得注意的是
- 值得一提的是
- 值得关注的是
- 不得不说
- 不可否认
- 毫无疑问
- 不容忽视
- 我们必须认识到
- 需要明白的是
- 需要指出的是
- 必须承认
- 让我们正视
- 客观来讲 / 客观地说
- 坦白说 / 老实说 / 说实话
- 在我看来（写第一人称文章时可保留，但密度别太高）

**判断方法**：把这句开头去掉，句子本身还成立吗？成立就删。

## 4. 互联网 / 商业黑话

2018 年之后中文互联网产出的造词，AI 训练数据里浓度极高。在公众号、产品稿、招聘文案里要尤其警惕。

### 4.1 动词类
赋能、抓手、打通、对齐、串联、闭环、跑通、沉淀、复盘、拉齐、拉通、收敛、发散、聚焦、辐射、引爆、撬动、激活、释放、加持、加码

### 4.2 名词类
颗粒度、心智、底层逻辑、链路、漏斗、增长飞轮、第一性原理、信息差、认知差、势能、势位、生态位、护城河、势在必行、势能差、心智占领、用户画像、价值传递、深度链接、生态构建、护城河

### 4.3 形容/状态
高维、降维、垂直、横向、纵深、立体、生态、生态化、规模化、平台化、垂类、长尾、头部、腰部、尾部、抗周期、反脆弱

### 4.4 流行黑话
卷、内卷、躺平、破圈、出圈、入局、破局、出海、上岸、躬身入局、风口、赛道、解决方案、痛点、爽点、痒点、刚需、强需求、弱需求

> **判断标准**：能不能用普通人听得懂的词替代？能就替代。  
> 例：「赋能销售团队」→「让销售卖得更顺」「给销售配工具」  
> 例：「打通用户心智」→「让用户记住我们做什么」

## 5. 程度副词 / 虚词泛滥

中文 AI 的"刷量词"，删了几乎不损失任何信息。

### 5.1 程度副词
非常、十分、极其、相当、特别、格外、尤其、极度、超级、巨、贼、好、很（每千字不超过 5 个"很"）

### 5.2 模糊语气
真的、确实、的确、实在、其实、本质上、说白了、本来、本身、原本

### 5.3 虚化量词
一种、某种、一定的、相应的、一定程度上、在某种意义上、一类、一系列、一系列的

### 5.4 名词化动词搭子（删搭子保留动作）

| 搭子 | 改成 |
|---|---|
| 进行 + N | 直接动 |
| 实现 + N | 直接做 |
| 做出 + N | 做 |
| 采取 + N | 具体动作 |
| 加以 + N | 直接动词 |
| 给予 + N | 给 |
| 提供 + N | 给 / 提供（少用） |
| 起到 + 作用 | 起作用 / 直接说做了什么 |
| 发挥 + 作用 | 同上 |
| 形成 + 局面 / 态势 | 写出具体形成了什么 |

> 例："对算法进行优化" → "优化算法"  
> 例："对问题加以解决" → "解决问题" / "把问题解决了"  
> 例："起到了一定的推动作用" → "推动了 XX" / 数据化

## 6. 元评论（自我指涉）

AI 喜欢预告自己要做什么、解释自己为什么这么写。这些都该删——读者要的是内容，不是导览。

- 让我们来探讨 / 让我们来分析 / 让我们一起 / 让我们看看
- 接下来我将 / 接下来我们将 / 下面我们来看
- 在本文中 / 本文将 / 这篇文章会
- 以下是几点 / 以下几个方面 / 下面分几点讲
- 让我为你详细讲解 / 让我来告诉你
- 我会从 XX 入手 / 我会先讲 XX 再讲 XX
- 在开始之前 / 在进入正题之前
- 说到这里 / 讲到这里
- 回到正题 / 言归正传（除非确实跑题了）
- 我想说的是 / 我想表达的是

**原则**：直接说事，不要解释"我接下来要说什么"。

## 7. 鸡汤收尾 / 抒情扩散

AI 段尾癖好。看到这些词在段末，删整句。

### 7.1 直接金句
- 愿你 / 愿我们 / 愿每一个 XX
- 未来可期 / 道阻且长 / 行而不辍
- 让我们一起 / 共创美好 / 携手前行 / 同行
- 你值得拥有 / 你不是一个人
- 与 XX 同行 / 向阳而生 / 奔赴山海
- 不负韶华 / 不负 XX
- 心之所向 / 素履以往
- 山高水长 / 来日方长

### 7.2 句式化升华
- "这，就是 XX 的力量"
- "这，就是 XX 的意义"
- "唯有 XX，方能 XX"
- "因为 XX，所以 XX 才显得弥足珍贵"
- "在这个 XX 的时代，我们更需要 XX"
- "或许，XX 才是真正的 XX"
- "XX，永远在路上"
- "致敬每一位 XX"

### 7.3 假深沉
- 人生如逆旅 / 岁月如歌 / 时光如水
- 三千繁华 / 一念之间
- 路虽远，行则将至

**判断标准**：这句话能套到任何主题上吗？能就删。

## 8. 排比强化型套话（句尾结构）

不是所有排比都坏，但 AI 用的排比有固定模板。

- "是 XX，是 XX，更是 XX"
- "是一种态度，是一种选择，是一种生活方式"
- "不仅是 XX，还是 XX，更是 XX"
- "既是 XX，也是 XX，又是 XX"
- "为了 XX，为了 XX，也为了 XX"
- "我们要 XX，我们要 XX，我们更要 XX"

**改写策略**：保留三句里最有力的一句，其余删；或者改成不同句式说出。

---

## 9. 假亲切称呼

AI 模仿"亲民"内容时高频使用。除非确认是面向私域社群的特定场景，否则一律删。

- 亲爱的读者 / 亲爱的朋友
- 朋友们 / 各位朋友
- 家人们（极强 AI / 直播话术指纹）
- 宝子们 / 宝宝们
- 各位看官 / 列位
- 小伙伴们（编程/技术博客慎用，已被 AI 滥用）
- 各位老铁
- 你我皆是 XX / 我们都是 XX

---

## 10. 价值判断饱和词

形容词不要花钱不要钱地撒。这些词每出现一次都要问"能不能换成具体的"。

- 重要、关键、核心、必不可少、不可或缺、举足轻重
- 显著、明显、巨大、深远、深刻、深邃
- 优秀、卓越、出色、突出、杰出
- 强大、有力、扎实、稳健、稳固
- 全新、崭新、焕然一新、耳目一新
- 璀璨、夺目、闪耀、绚烂
- 极致、终极、完美、无可挑剔

> 例："带来了显著的提升" → "加载从 4.2 秒降到 0.8 秒"  
> 例："是核心能力" → "如果没这个，整个流程跑不通"

## 11. 四字成语滥用

成语本身无罪，但 AI 文本里成语密度异常高，且偏爱一组特定的"正能量成语"。同段不超过 1 个，全篇不堆砌。

**高频 AI 成语**（看到要警觉）：
博大精深、源远流长、生机勃勃、欣欣向荣、蓬勃发展、日新月异、与时俱进、推陈出新、独树一帜、独具匠心、精益求精、匠心独运、锦上添花、画龙点睛、点石成金、点亮未来、照亮前行、引领潮流、勇立潮头、披荆斩棘、攻坚克难、风雨同舟、砥砺前行、笃行不怠

**判断**：如果连续两句话各用一个四字成语，删一个。如果一句话里两个成语并列，删一个。

## 12. 比喻库存（被 AI 用烂的比喻）

这些比喻本身可能曾经鲜活，但被训练数据里反复使用，已经是套话。

- 像一把双刃剑
- 好比一座桥梁 / 是连接 XX 的桥梁
- 如同 XX 的灯塔 / 指明方向
- 是 XX 的引擎 / 发动机
- 是 XX 的基石 / 基础 / 根基
- 是 XX 的催化剂
- 像一面镜子，映照出
- 如同一棵大树 / 一颗种子
- 是一场马拉松 / 不是百米冲刺而是马拉松
- 是 XX 的密码 / 钥匙
- 是 XX 的天花板 / 上限

**改写方向**：如果一定要比喻，找一个跟内容紧密相关、读者不太可能见过的比喻；找不到就直接说，不比喻。

## 13. 英译洋腔词（AI 直译腔）

英文大模型的中文输出带着一股"翻译腔"。这些词在英文 AI 文本里是高频套话，被直译进中文后同样刺眼。中文母语者日常几乎不这么说话。

| 洋腔词 | 英文原型 | 改成 |
|---|---|---|
| 深入研究 / 深入探讨 | delve into | 研究 / 看 / 查 |
| 见证（"XX 是 XX 的见证"） | testament to | 说明 / 证明 / 直接说事 |
| 格局 / 图景（"XX 的格局"） | landscape | 现状 / 情况 / 局面 |
| 强调（"这强调了 XX"） | underscore / highlight | 说明 / 表明（或删） |
| 错综复杂 / 纷繁复杂 | intricate / complex | 复杂 / 乱（或写出具体怎么复杂） |
| 充满活力 / 生机勃勃 | vibrant | 热闹 / 有人气（或删） |
| 无缝 / 无缝衔接 | seamless / seamlessly | 顺 / 不卡顿 / 直接接上 |
| 增强 / 提升（万能动词） | enhance | 写出具体改了什么 |
| 利用（"利用 XX 来 XX"） | leverage / utilize | 用 |
| 促进 / 推动（空转） | foster / facilitate | 帮 / 让（或写具体动作） |
| 培养（"培养一种 XX"） | cultivate | 养成 / 练（或删） |
| 拥抱（"拥抱变化 / 拥抱 AI"） | embrace | 接受 / 适应 / 开始用 |
| 解锁（"解锁新能力"） | unlock | 学会 / 多了一项（或删） |
| 至关重要（句首万能锤） | crucial / pivotal | 写出对谁重要、不重要会怎样 |

### 13.1 英译 AI 口头禅（语气直译）

英文 AI 的固定腔调被直译进中文，本土中文里没人这么开口：

- 我必须诚实地说 / 老实讲（= I must be honest）
- 这个问题是真实存在的（= this is a real concern）
- 让我们深入了解一下（= let's dive in）
- 重要的是要注意（= it's important to note）
- 值得深思 / 引人深思（= worth pondering）
- 在 XX 的宏大图景中（= in the grand scheme of things）

**判断**：把句子读出声，像不像一个中国人会当面对你说的话？不像，多半是直译腔，删或改成口语。

## 14. 无名归因（黄鼠狼话术）

给观点找一个看不见的"权威"背书，但拿不出实名来源。英文叫 weasel words。AI 用它给平庸结论镀一层可信度。

- 专家认为 / 专家指出 / 有专家表示
- 研究表明 / 研究显示 / 多项研究证明
- 数据显示 / 数据告诉我们（无出处时）
- 有人说 / 人们普遍认为 / 业内普遍认为
- 众所周知 / 大家都知道
- 据悉 / 据了解 / 相关人士透露（非新闻稿场景）

**判断标准**：能不能补上具体的人名、机构、论文标题、年份？

- 能 → 补上（"2024 年斯坦福 Diao 等人的研究发现……"）
- 不能 → 这是糊弄，删掉整句，或改成第一人称判断（"我自己的经验是……"）

不要用一个虚构的"专家"替自己背书。要么有据，要么署上自己的名。

## 15. 评级悬念词（先评级再陈述）

AI 转述材料时的惯性：每讲一个点，先给它盖一个"这有多惊人"的戳，再说内容。单个看是修辞，通篇段段用就是转写腔的指纹——尤其技术文、评测文、源码解读最高发。

### 15.1 "最 XX 的是"悬念开场

- 最反直觉的是 / 最反直觉的发现是
- 最容易让人栽跟头的是 / 最容易踩坑的是 / 最容易被误解的是
- 最值得注意的是 / 最值得记住的是 / 最值得警惕的是
- 最出乎意料的是 / 最让我意外的是
- 最隐蔽的坑是 / 最微妙的地方是

### 15.2 弱化版评级前缀

- 有意思的是 / 有趣的是 / 好玩的是
- 令人惊讶的是 / 令人意外的是
- 更微妙的是 / 更值得警惕的是 / 更隐蔽的是
- 值得玩味的是 / 耐人寻味的是
- 反直觉（做形容词单用："一个反直觉的设计"）

**判断标准**：把评级前缀删掉，直接从事实开句，段落损失了什么？通常什么都没损失——惊讶应该由事实本身触发，不是由"这很惊讶"的预告触发。

**配额**：一篇文章里"最 XX 的是"最多留 1 个，放在全文真正最重的那个点上。其余全删，直接陈述。

> **原**：最反直觉的发现是"写了优雅降级但没人用"。`attention.py` 里封装了……  
> **改**：`attention.py` 里封装了一个带 SDPA 兜底的分发函数，但 DiT 主干没有用它，直接调了最底层的 `flash_attention()`。（事实自己会说话）

## 使用说明

这份清单不是要"逐词消灭"，是要建立**敏感度**。

- 写作时：不必边写边查
- 改写时：从清单 1 → 5 → 7 → 10 顺序扫一遍（命中率最高的几类）
- 校稿时：用搜索功能批量找 "首先 / 进行 / 一种 / 让我们 / 非常 / 综上"，每个都问一遍要不要

清单会持续更新。新增 AI 套话以"在三篇不同来源的 AI 输出里都出现过"为收录标准。
